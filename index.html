My Projects

Here are a few of my recent projects. Each one showcases my skills in front-end and back-end development.

Project 1

The Ration Raksha App (RRA) is a mini-project designed to digitalize the Public Distribution System (PDS). This innovation aims to streamline the process of ration delivery and management, ensuring greater transparency and efficiency. By providing a digital platform, RRA helps to simplify access to essential commodities for beneficiaries and offers a more secure and accountable system for managing food distribution.

Project 2

My main project, Companion Core, is a true passion of mine. It's an innovative system designed to operate a PC using nothing but voice commands and hand gestures, completely replacing the need for a mouse and keyboard. This project allowed me to dive deep into exciting fields like natural language processing and computer vision, where I gained hands-on experience in developing a more intuitive and accessible experience.

Project 3

As an Indian computer science student, I wanted to create a project that truly resonated with me. Bharat Yatra is the result—a digital journey through my homeland. This tourism website goes beyond a simple guide; it's a showcase of my skills in web development, design, and data management. It combines stunning visuals with detailed travel information, offering a virtual passport to India's diverse landscapes and cultures.

Project 4

My Portfolio Website: This is more than just a place to showcase my work—it's a project in itself. I designed and developed this site from the ground up using HTML, CSS, JavaScript. It demonstrates my skills in creating a clean, responsive, and visually appealing user experience.

Project 5

Interactive Rock-Paper-Scissors: Developed using HTML, CSS, and JavaScript, this project was a hands-on learning experience in DOM manipulation. It's a simple game that demonstrates a fundamental skill in front-end development: connecting user actions to changes on the webpage, such as updating scores and displaying game results.

Project 6

Employee Management System (EMS): This project is a front-end web application built using HTML, CSS, and JavaScript. It serves as a practical demonstration of my ability to build a data-driven interface, allowing users to add, view, edit, and delete employee records. This project highlights my skills in DOM manipulation and managing application state on the client side.
